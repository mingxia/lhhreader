# lhhreader

最舒适、最优雅的连环画阅读器（电脑及平板端优先适配）。

## 当前方向

- 技术栈：Next.js + React
- 阶段目标：先完成阅读器界面与核心阅读交互
- 数据方案：使用本地/服务器 JSON 文件，不引入数据库

详见：`lianhuanhua-reader-features.md`

## 阅读器界面原型（首版）

已新增一个可直接打开的阅读器界面原型，覆盖当前阶段的核心交互：

- 双页/单页切换
- 适应宽度/适应高度
- 缩放（50%~250%）与 100% 还原
- 全屏切换（按钮与 `F` 快捷键）
- 键盘翻页（`←` / `→` / `空格` / `Shift + 空格`）
- 左中右热区点击翻页/切换 UI
- 顶部与底部浮层自动淡出（沉浸模式）
- 非全屏下的推荐卡片区

运行方式（无需构建）：

1. 直接在浏览器打开 `reader/index.html`。
2. 或在仓库根目录执行：`python3 -m http.server 4173`，然后访问 `http://localhost:4173/reader/`。
